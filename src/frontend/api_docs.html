Anomaly Prediction

POST
/predict

Submit login data to detect anomalies.

Request Body

{
  "user_id": "string",
  "timestamp": "string (ISO 8601 format)",
  "ip_address": "string",
  "geo_location": "string",
  "device_id": "string",
  "login_hour": "integer (0-23)",
  "is_new_device": "integer (0 or 1)",
  "bytes_in": "integer",
  "bytes_out": "integer",
  "success": "integer (0 or 1)"
}

Response

{
  "anomaly": "integer (0 or 1)",
  "probability_normal": "float",
  "probability_anomaly": "float"
}

Example Usage

Using curl to make a prediction:

curl -X POST http://localhost:5000/predict \
  -H "Content-Type: application/json" \
  -d '{
    "user_id": "U12345",
    "timestamp": "2025-09-15T14:30:00",
    "ip_address": "192.168.1.100",
    "geo_location": "United States of America",
    "device_id": "DF_1234",
    "login_hour": 14,
    "is_new_device": 0,
    "bytes_in": 102400,
    "bytes_out": 51200,
    "success": 1
  }'
